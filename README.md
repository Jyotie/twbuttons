### Building

#### 0.0.1 Version
required [node.js](http://nodejs.org/) & [bower](http://bower.io/) & [grunt](http://gruntjs.com/getting-started)

```shell
    $ git clone https://github.com/onokumus/buttons.git yourfoldername
    $ cd yourfoldername
    $ npm install
    $ npm run build
```
